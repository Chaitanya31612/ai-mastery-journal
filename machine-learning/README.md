# Machine Learning

Hands-on machine learning track designed for a software engineer moving from application development to practical ML understanding.

## Folder Structure

```text
machine-learning/
├── docs/
│   ├── general/      # Core ML concepts and theory with code examples
│   └── projects/     # Step-by-step project learning guides
├── code/
│   ├── from-scratch/ # Numpy-only implementations
│   ├── sklearn/      # Production-style sklearn workflows
│   ├── projects/     # End-to-end project implementations
│   ├── utils/        # Reusable helpers
│   ├── datasets/     # Local datasets or download scripts
│   └── notebooks/    # Exploration and visualization
└── plan/
    └── 01-ml-intermediate-roadmap.md
```

Start with the roadmap: `plan/01-ml-intermediate-roadmap.md`.
